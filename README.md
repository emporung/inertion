Inertia Tensor Visualizer

Программа для вычисления тензора инерции и визуализации эллипсоида инерции твёрдого тела.

- Вычисление тензора инерции для 5 тел: куб, шар, параллелепипед, цилиндр, конус.
- Определение главных моментов инерции и главных осей.
- Вычисление момента инерции относительно произвольной оси вращения.
- 3D-визуализация эллипсоида инерции, тела и оси вращения.
